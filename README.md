**Cargo subcommand that applies predefined [Cargo profile](https://doc.rust-lang.org/cargo/reference/profiles.html)
templates to get you up to speed quickly.**

# Installation

```bash
$ cargo install cargo-wizard
```

## Features

- Modify Cargo.toml
- Modify `.cargo/config.toml`
